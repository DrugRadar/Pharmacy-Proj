Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Addresses

Update the specified address of a client

Example request:
curl --request PUT \
    "http://localhost:8000/api/client/1/address/7" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"street_name\": \"ut\",
    \"building_number\": \"inventore\",
    \"floor_number\": \"et\",
    \"flat_number\": \"eligendi\",
    \"is_main\": true
}"

const url = new URL(
    "http://localhost:8000/api/client/1/address/7"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "street_name": "ut",
    "building_number": "inventore",
    "floor_number": "et",
    "flat_number": "eligendi",
    "is_main": true
};

fetch(url, {
    method: "PUT",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Register a new client.

Example request:
curl --request POST \
    "http://localhost:8000/api/client/register" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=John Doe" \
    --form "email=john.doe@example.com" \
    --form "gender=male" \
    --form "password=secret" \
    --form "date_of_birth=1999-03-30" \
    --form "mobile_number=huwyerkanzj" \
    --form "national_id=05072619594761" \
    --form "phone=+1234567890" \
    --form "avatar_image=@C:\Users\radwa\AppData\Local\Temp\phpEEF4.tmp" 
const url = new URL(
    "http://localhost:8000/api/client/register"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('name', 'John Doe');
body.append('email', 'john.doe@example.com');
body.append('gender', 'male');
body.append('password', 'secret');
body.append('date_of_birth', '1999-03-30');
body.append('mobile_number', 'huwyerkanzj');
body.append('national_id', '05072619594761');
body.append('phone', '+1234567890');
body.append('avatar_image', document.querySelector('input[name="avatar_image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
